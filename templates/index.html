How It Works

1. Compression

Files are optionally compressed using Huffman encoding to reduce size.

2. Encryption

Data can be encrypted for security before converting to DNA.

3. Error Correction

Reed-Solomon error correction ensures data integrity.

4. DNA Encoding

Final data is encoded using the four DNA nucleotides: A, T, G, C.
